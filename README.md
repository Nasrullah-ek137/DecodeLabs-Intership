# E-Commerce Exploratory Data Analysis (EDA)

## 📌 Internship Project - DecodeLabs

This project was completed as part of the **DecodeLabs Data Analytics Internship Program**. The main objective of this project was to perform **Exploratory Data Analysis (EDA)** on an E-Commerce dataset to uncover business insights, customer purchasing behavior, sales trends, and product performance using Python.

The project focuses on transforming raw data into meaningful insights through data cleaning, statistical analysis, visualization, and correlation analysis.

---

# 🚀 Project Objectives

- Understand customer purchasing patterns
- Analyze sales trends and product performance
- Perform data cleaning and preprocessing
- Detect outliers and anomalies
- Generate business insights using visualizations
- Apply statistical analysis techniques on real-world data
